Conceptos

¿Qué es el sistema binario?

Es un sistema de numeración basado en dos dígitos(0 y 1). Es el idioma base de las computadoras para comunicarse y almacenar información. Datos como texto, imágenes y sonido se transforman en código binario para poder ser procesados.

¿Qué es el complemento a 2?

Método para representar números negativos: se invierten los bits y se suma 1. El complemento a dos es un sistema de representación numérica binaria utilizado por la mayoría de las computadoras modernas para codificar números enteros positivos y negativos. Permite realizar operaciones aritméticas eficientes y simplifica los circuitos lógicos en los sistemas digitales.

¿Qué es overflow?

En la arquitectura de computadoras es la condición en la que un resultado aritmético o datos superan la capacidad máxima de almacenamiento de un registro, memoria o tipo de dato, provocando corrupción de datos o errores en el sistema. Se produce cuando un valor excede el límite del contenedor.

¿Qué hace una ALU?

Una ALU(Unidad Aritmético-Lógica) operaciones aritméticas y lógicas como suma, AND, OR, XOR y NOT. Funciona como el un cerebro matemático que procesa instrucciones, recibe operandos, realiza cálculos y devuelve el resultado.
